# FreeBSD pNFS Demo 

## About

An unattended pNFS demo on four virtualized FreeBSD nodes, mds, ds0, ds1, and client. It took 4m56.369s if the FreeBSD ISO file was downloaded.

## Run
```
make -j
```

## Test
Tested on Arch Linux on 2023-04-02.
